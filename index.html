Welcome to MQWeb

Warning! This website is still a work in progress ...

Administrating your Websphere MQ environment using your favorite browser. That's the purpose of MQWeb. You can use the build-in web pages, but you can also write your own web pages with any server-side language and only use the JSON objects that are returned by the MQWeb daemon.

HTTP Daemon

The core of MQWeb is an HTTP daemon written in C++ using the POCO project. Based on the URI, the daemon will return HTML or JSON. Read more on which features are already available.
