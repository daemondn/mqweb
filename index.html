Welcome to MQWeb

Warning! This website is still a work in progress ...

Administrating your Websphere MQ environment using your favorite browser. That's the ultimate goal of MQWeb. You can use the build-in web pages, but you can also write your own web pages with any server-side language and only use the JSON objects that are returned by the MQWeb daemon.

HTTP Daemon

The core of MQWeb is an HTTP daemon written in C++ using the POCO project. Based on the URI, the daemon will return WebSphere MQ information in HTML format or as JSON objects. Read more on which features are already available.

Why MQWeb ?

  • It's fast

    It's written in C++ and uses the MQ API's directly.

  • It's easy to install

    One executable, one configuration file. There is no need to install an application server, JVM, ...

  • It's easy to adopt

    You can use the build-in HTML pages, write your own HTML pages, use Asynchronous JavaScript and JSON (AJAJ) and integrate it in your own websites.
